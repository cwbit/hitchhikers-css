1. [Centering an Image]()

##### Centering an Image
Apply the following style directly to the `<img>`

```
	margin-right: auto;
	margin-left: auto
```

or create the following css class and assign it to the `<img>`

```
.img-centered{
	margin-left: auto;
	margin-right: auto;
}
```
